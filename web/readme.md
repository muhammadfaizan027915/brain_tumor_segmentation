**Run Project**

***Docker Installation***
1. Download the docker desktop from [Download](https://docs.docker.com/desktop/setup/install/windows-install/)
2. Install the docker desktop
3. Start the docker desktop


***Command:***
1. After docker desktop installation go to ```web``` folder.
2. Run the following command. ```docker compose up --build```


***Instructions***
1. On dependency installation create requirements.txt
2. Rebuild the run the docker compose
